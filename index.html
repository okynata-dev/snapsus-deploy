FAQ

Questions you'd ask if we sat down.

How accurate are historical snapshots?

We resolve the block nearest your chosen timestamp via DefiLlama (≈30-second accuracy) and replay every transfer up to that block. The output reflects ownership exactly as it stood. Block numbers are recorded in the receipt so you can verify externally.

Is my data stored anywhere?

No accounts, no database. CSVs and JSON archives live only on your machine. The Cloudflare edge caches API responses for a few minutes to reduce load — that's it. Shared snapshot links you publish via "Get a shareable link" stay in KV for one year.

What if my collection was deployed via Manifold or Foundation?

Factory-deployed collections are picked up by a secondary pass: we list contracts your wallet appears in, then check whether you're the OpenSea-indexed creator or contract deployer for each. Coverage isn't perfect — if anything's missing, paste the contract address directly.

How are smart contracts excluded?

Toggling "Exclude smart contracts" runs an eth_getCode batch over every holder on Ethereum. Anything with bytecode (Conduit, Blur Pool, gem.xyz, safes, vaults, routers) drops from the list.

Does it cost anything?

No. The site is free and we don't sell data. Heavy abuse is rate-limited per IP; for sustained high-volume usage get in touch.
