How an agent calls a tool

  1. Discover it in OpenSea's onchain registry (or load it directly).
  2. POST the endpoint with a JSON body. With no payment it returns 402 + the price.
  3. The agent signs an x402 USDC payment and retries — it gets the JSON result back. Settlement is gasless via the facilitator.
